# Batch Costing Calculator

Formula Raw batch costing calculator - Real-time profit & margin analysis for all products.

## Features

✅ **Real-time Cost Calculations** - Instant profit & margin analysis  
✅ **Product Selection** - Choose from all Formula Raw SKUs  
✅ **Batch Yields** - Calculate costs for any batch size  
✅ **Responsive Design** - Works on desktop, tablet, mobile

## Deployment

This app is deployed on **Vercel** - the easiest way to deploy Next.js apps.

### Quick Deploy

1. Go to https://vercel.com/new
2. 2. Import this GitHub repo
   3. 3. Click "Deploy"
      4. 4. Done! Your app is live 🚀
        
         5. ## Usage
        
         6. 1. **Select a product** from the dropdown
            2. 2. **Enter batch yields** (number of units produced)
               3. 3. **Set retail price** per unit
                  4. 4. **View calculations instantly** - cost, profit, margin percentage
                    
                     5. ## Tech Stack
                    
                     6. - **Frontend**: React 18 with Tailwind CSS
                        - - **Backend**: Next.js API Routes
                          - - **Deployment**: Vercel
                            - - **Database**: In-memory storage (upgradeable to Supabase)
                             
                              - ## Formula
                             
                              - ```
                                Cost per Unit = (Ingredient Cost × 1.10) / Batch Yields
                                Profit = Retail Price - Cost per Unit
                                Margin % = (Profit ÷ Retail Price) × 100
                                ```
